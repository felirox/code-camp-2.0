Features

Completely Online

Gone are the days when encrypting a file required multiple applications to be installed on both the host and receiver computers. It can be used anywhere, so long as you have internet

Same Format

The format of the encrypted file is identical to that of the original. Upload encrypted files to locations that only accept specific file types.
Keep the file hidden in plain sight.

This application is a Progressive Web Application. This makes it possible to install and use this app easily through both mobile and computer with 0 local storage utilized.
